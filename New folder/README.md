# Bonagiri Manoj Kumar - Portfolio

A clean, modern personal portfolio website inspired by professional design references.

## Features

- **Home** - Hero section with name, title, and call-to-action buttons
- **About** - Personal intro + Education timeline (SR University, Narayana Junior College, Sri Siddartha High School Husnabad)
- **Skills** - Technical skills grid (C, Python, HTML/CSS, JavaScript, SQL, etc.)
- **Services** - Web Designing & Backend Development
- **Projects** - Student Helpdesk System, AI Health Chatbot
- **Contact** - Contact form + Email, LinkedIn, GitHub links

## Setup

1. Open `index.html` in a browser, or run a local server:
   ```bash
   npx serve .
   ```

2. **Add your profile photo**: Place an image named `profile.jpg` in the same folder as `index.html`

3. **Update contact info** in `index.html`:
   - Replace `your.email@example.com` with your email
   - Add your LinkedIn profile URL
   - Add your GitHub username

4. **Add project links**: Update the `href="#"` in the project cards with your GitHub or live demo URLs

5. **Update social links**: Add your website, LinkedIn, and email URLs in the hero section

## Customization

Edit `index.html` to change text content. Edit `styles.css` to modify colors (teal accent is defined in `:root`) and layout.
